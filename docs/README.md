# Documentation Layout

- `plans/active/`: plans that are still driving current work.
- `plans/archive/`: completed or superseded implementation plans kept for history.
- `runbooks/`: operational instructions for running or reproducing workflows.
- `archive/legacy-paper/`: older manuscript assets kept only for reference; this is separate from the current work.

The root-level summary file captures current project state, while dated plan files track execution history.
